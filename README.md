# Bouwplanning — reparatieplanning Kwekerij Baas

De hele app is één bestand: `public/index.html`. Alle data loopt live via de beveiligde
Supabase-API; dit repo bevat dus geen geheimen en geen data — alleen de pagina.

## Nieuwe versie van de app publiceren
1. Vervang `public/index.html` door de nieuwste `Reparatieplanning - Jarno.html`
   (uit OneDrive-map "Hagelschade 2026/Reparatie planning", hernoemen naar index.html).
2. Commit + push naar main → Webflow Cloud deployt automatisch.

## Links (na koppeling domein baaskwekerij.nl)
- Jarno (direct binnen, geen inlogscherm): https://baaskwekerij.nl/#kas2026
- Testen als Dieter: https://baaskwekerij.nl/#kas2026/Dieter
